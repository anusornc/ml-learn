Reinforcement Learning (การเรียนรู้เสริมแรง)

เอเจนต์เรียนรู้ในสิ่งแวดล้อมจำลอง กระบวนการตัดสินใจ MDP และ Q-Learning

1 การเรียนรู้เสริมความแกร่ง (Reinforcement Learning)

การเรียนรู้เสริมความแกร่ง (Reinforcement Learning) แตกต่างจากแนวทางการเรียนรู้แบบอื่น ๆ เพราะโมเดลไม่ได้ถูกสอนด้วยคู่คำตอบสำเร็จรูป แต่จะใช้วิธีให้โมเดลจำลองตัวตนเป็นตัวแทนกระทำ **(Agent)** ลงไปทดลองแก้ปัญหาในสภาพแวดล้อม **(Environment)** จริง และเรียนรู้ผ่านผลการสะท้อนกลับในรูปของรางวัล **(Reward)** หรือบทลงโทษเชิงคะแนนลบ

วงจรการปฏิสัมพันธ์ใน RL (RL Loop)
[ Agent (ตัวแทน) ]
      │   Action (กระทำ)
      ▼
[ Environment (สภาพแวดล้อม) ]
      │   Reward (รางวัล) / New State (สถานะใหม่)
      └────────────────────────► (กลับไปที่ Agent)

2 กระบวนการตัดสินใจแบบมาร์คอฟ (Markov Decision Process หรือ MDP)

ปัญหาใน Reinforcement Learning ส่วนมากจะถูกจำลองให้อยู่ภายใต้กรอบของ **MDP** ซึ่งเป็นโมเดลคณิตศาสตร์สำหรับระบุสภาพแวดล้อม โดยประกอบด้วยองค์ประกอบ:

  • $S$ (State): เซตของสถานะที่เป็นไปได้ทั้งหมดในสภาพแวดล้อม
  • $A$ (Action): เซตของการตัดสินใจทำกิจกรรมที่ Agent สามารถทำได้
  • $T(s, a, s')$ (Transition Probability): โอกาสที่จะเปลี่ยนจากสถานะ $s$ ไปยัง $s'$ เมื่อตัดสินใจทำ $a$
  • $R(s, a, s')$ (Reward): ค่าคะแนนผลตอบแทนที่ได้รับ
  • $\gamma$ (Discount Factor): ตัวคูณลดทอนคุณค่าของรางวัลในอนาคต (มีค่าระหว่าง 0 ถึง 1) เพื่อประเมินน้ำหนักรางวัลในปัจจุบันให้สูงกว่าการรอคอยในอนาคต

3 ขั้นตอนวิธี Q-Learning

**Q-Learning** เป็นอัลกอริทึมประเภท **Model-Free Reinforcement Learning** ที่มุ่งคำนวณและประเมินค่าฟังก์ชันความคุ้มค่าของการกระทำที่เรียกว่า **Q-Value ($Q(s, a)$)** โดยจะจัดเก็บไว้ในตาราง **Q-Table** เพื่อระบุว่าในสถานะ $s$ ใด ๆ การเลือกทำ $a$ จะคุ้มค่าที่สุดในระยะยาว

สมการการอัปเดตค่า Q-Value (Q-Table Update)

$$Q(s, a) \leftarrow Q(s, a) + \alpha \left( r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right)$$

ความหมายของสัญลักษณ์:

  • $\alpha$ (Learning Rate): อัตราการเรียนรู้เพื่อเปลี่ยนค่าเดิม
  • $r$: รางวัลที่ได้รับทันทีในตานั้น
  • $\max_{a'} Q(s', a')$: ค่าประเมินสถานการณ์ Q-Value ที่ดีที่สุดในอนาคตเมื่อเข้าสู่สถานะใหม่ $s'$
